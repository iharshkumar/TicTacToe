# TicTacToeGame

A simple browser-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Files
- index.html — game UI
- app.js — game logic
- style.css — styles

## Run
Open [index.html](index.html) in your browser, or serve the folder with a static server:


## Play
- Click a cell to place your mark.
- Players alternate turns; the game detects wins and draws.
- Refresh the page to restart.
